# 🚀 TaskFlow Lite Pro

A modern, responsive, and feature-rich Task Management Application built using **HTML, CSS, and JavaScript**.

TaskFlow Lite Pro helps users organize daily tasks efficiently with task creation, editing, completion tracking, filtering, searching, dark mode support, and local storage persistence.

---

## 🌟 Project Highlights

* Modern Glassmorphism User Interface
* Fully Responsive Design
* Dark & Light Theme Support
* Real-Time Task Statistics
* Client-Side Data Persistence
* Clean and Modular JavaScript Code
* Beginner-Friendly and Easy to Use

---

## 📌 Features

* ✅ Add New Tasks
* ✅ Edit Existing Tasks
* ✅ Delete Tasks
* ✅ Mark Tasks as Complete/Incomplete
* ✅ Search Tasks Instantly
* ✅ Filter Tasks (All / Active / Completed)
* ✅ Task Statistics Dashboard
* ✅ Dark Mode Support
* ✅ Local Storage Persistence
* ✅ Responsive Design
* ✅ Modern Glassmorphism UI

## 🎯 How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in any modern browser.

No installation or additional dependencies are required.

---

## 💾 Data Storage

TaskFlow Lite Pro uses the browser's Local Storage API.

Benefits:

* Tasks remain saved after refreshing the page.
* No backend or database required.
* Fast client-side performance.
